Nebula Infinite

Overview

Nebula Infinite is my first attempt at making an infite runner and a mobile game. Within this game the player would simply press and hold the screen to fly upwards and let go to fall downwards. The player's goal is to surivive as long as they can while dodging asteroids.

Features

  • Score that increases and only stops when the player dies
  • Game gets harder as the score gets higher

Project Design Goals

  • Design for a platform I never designed for before

Tech

  • Platform: Windows PC
  • Engine: Unity
  • Language: C#
  • Tools Used: Unity, Piskel
  • Team Size: 1
  • Role: Game Designer, Gameplay Programmer, System Programmer, Sound Designer
