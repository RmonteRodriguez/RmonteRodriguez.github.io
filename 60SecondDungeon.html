60 Second Dungeon

Overview

60 Second Dungeon is a top down shooter where I really wanted to experiment more with creating AI and to mess with procedual generation. This game has a few enemy types and even some bosses. The games main goal is to clear each floor of the duengeon within 60 Seconds or the player loses.

Features

  • Top down player controller that allows the player to shoot enemies
  • Handfull of different enemies with different behaviors
  • Variety of bosses that appears every few floors
  • Procedual generated floors with an exit that allows players to go to next floor
  • Only 60 seconds per floor before game over

Project Design Goals

  • Design a game with the theme of 60 seconds
  • Design enemy AI
  • Make a player controller for a different medium I have not done before

Tech

  • Platform: Windows PC
  • Engine: Unity
  • Language: C#
  • Tools Used: Unity
  • Team Size: 1
  • Role: Game Designer, Gameplay Programmer, System Programmer, Sound Designer
